Mis Proyectos

Enviador de Facturas Automatizado

Aplicación para enviar facturas PDF por email de forma masiva, automatizando el mapeo de destinatarios desde Excel y reduciendo errores manuales. Resuelve discrepancias en formatos de nombres y emails, con trazabilidad completa mediante logs y reportes.

Tecnologías clave:

  • GUI con Tkinter (interfaz, treeview, logs).
  • Multihilo (threading) para envíos no bloqueantes.
  • Pandas/Openpyxl para procesamiento de Excel multi-hoja.
  • Regex y Unicodedata para normalización de texto.
  • SMTPlib y email.mime para correos con adjuntos (Gmail TLS).

Editor de Facturas Automatizado

Herramienta de escritorio para generar facturas PDF desde plantillas Word, cargando datos de Excel y procesando lotes. Preserva formatos, calcula totales automáticamente y maneja numeración secuencial, resolviendo errores manuales y heterogeneidad de datos.

Tecnologías clave:

  • Tkinter para interfaz de usuario práctica con logs.
  • Pandas/Openpyxl/Xlwings para lectura robusta de Excel con fallbacks.
  • Python-docx y Lxml para reemplazo de marcadores preservando estilos.
  • Win32com para conversión a PDF y cálculos precisos con Decimal.
  • Procesamiento por lotes con reportes de éxitos/fallos.

Visor 3D de Normativa Urbanística (Galicia)

Demo para evaluar normativa urbanística (Ley 2/2016 Galicia) en mapas 3D, generando informes y volúmenes edificables. Integra chat IA para consultas y exporta modelos 3D, resolviendo análisis geolocalizados y visualización de impactos.

Tecnologías clave:

  • FastAPI/Python para backend con endpoints de evaluación y exportación.
  • FAISS y LLMs (GGUF) para chat IA con búsqueda semántica y fallback legal.
  • Three.js/MapLibre para visor 3D con carga OSM y extrusión dinámica.
  • CityJSON/GLTF/GLB para exportación de volúmenes 3D robusta.
  • CSV/Mock providers para datos municipales y heurísticas normativas.

Portfolio Profesional Interactivo

Sitio web personal con impacto visual, mostrando proyectos y certificados con UX dinámica. Incluye un "dragón" asistente para interacciones contextuales, resolviendo presentación estática y migración futura a React para escalabilidad.

Tecnologías clave:

  • HTML/CSS para layouts responsivos, gradientes y efectos 3D/hover.
  • JavaScript vanilla para gestión de estados y animaciones no intrusivas.
  • IntersectionObserver para comportamientos contextuales por sección.
  • Accesibilidad con lazy loading y cierre por teclado.
  • Estructura de datos centralizada para render dinámico de certificados.
